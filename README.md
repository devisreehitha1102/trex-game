# trex-game

This is the T-rex game which pops in any browser when there is no data in your mobile.
This is initiated by Google to engage the people in the browser even when there is no data.

# HOW TO PLAY

It is running game of a black and white T-rex
*Press space to jump
*Avoid the obstacles

# HOW TO RUN LOCALLY

Clone the repo and open the "index.html" in your browser
